# Number Guessing Game

# Description
- This game will test your skills on guessing a number between 1-100, and the player tries to guess it by typing a number and pressing submit. After each guess, the game tells the player if the number is too high, too low, or correct. It also shows how many guesses the player made.

# Features
- Submit button
- Attempt counter
- Box to type answer
- Gives a reply for every answer
